## 1. 커서 이동

+ **`^`**
    + 문장 맨 앞으로 이동
+ **`$`** 
    + 문장 맨 뒤로 이동
+ **`gg`**
    + 문서 맨 앞으로 이동
+ **`G`**
    + 문서 맨 아래로 이동


---

## 2. 편집

+ **`dd`**
    + 현재 줄 잘라내기
+ **`u`**
    + 실행취소(Undo)

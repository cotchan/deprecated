## 2. 자동완성
 
+ **`cmd` + `Shift` + `enter`**

## 3. extract method

+ **`ctrl` + `t`** (refactoring 관련: 여기안에 extract method가 있습니다.)

## 4. Create New test

+ **`cmd` + `shift` + `t`**

## 5. 이전에 실행했던 거 그대로 실행

## 6. 생성자와 Getter, Setter 생성

+ **`ctrl` + `enter`**
